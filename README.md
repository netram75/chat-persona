# Persona Chatbot

## Features
- 3 personas (Anshuman, Abhimanyu, Kshitij)
- Persona switching (resets conversation)
- Chat interface with typing indicator
- Prompt engineering techniques applied

## Setup
1. Add API key in backend/.env
2. Run backend:
   node server.js
3. Run frontend:
   npm start

## Tech Stack
- React
- Node.js (Express)

## Note
Fallback responses are used if API quota is unavailable.
Focus is on prompt engineering quality.
